# Copilot + GitHub Integration Plugin

Dieses Plugin integriert GitHub und Copilot in Obsidian.

## Features
- GitHub Repo Pull
- Copilot Chat (OpenAI-kompatibel)
- KI-gestützte Notizen
- BRAT-kompatibel

## Installation über BRAT
1. Obsidian öffnen
2. BRAT installieren
3. „Add Beta Plugin“
4. Repo-URL eingeben:

https://github.com/BlackHoleGER/obsidian-copilot-github

5. Aktivieren
